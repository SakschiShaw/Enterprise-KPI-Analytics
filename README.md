 Enterprise KPI Analytics
End-to-end Business Analyst project: BRD -Jira user stories - SQL KPIs -Power BI dashboards
 Transformation

 Project Overview
This project demonstrates a complete Business Analyst workflow:
Business Requirements - Jira Agile execution - SQL KPI analysis - Power BI dashboards.

 Business Problem
A global retailer needed centralized KPI dashboards to enable 20% faster executive decision-making during supply chain disruptions.

Tools Used
- Jira (Agile, User Stories, Sprints)
- SQL ( queries)
- Excel (Data preparation)
- Power BI (Executive dashboards)

 Key KPIs
- Total Revenue
- Monthly Sales Trends
- Active Customers (Churn Indicator)

 Outcome
Centralized KPI visibility enabled simulated 20% faster decision-making for leadership.
